1. Install Node.js LTS Version
First, download and install the latest LTS version of Node.js from the official website.

2. Initialize a New Node.js Project
Open your terminal and create a new directory for your project:

mkdir my-node-project
cd my-node-project

Initialize a new Node.js project:

npm init -y

This will create a package.json file with default settings.

3. Set Up TypeScript in the Project
Install TypeScript and the necessary types:

npm install typescript @types/node --save-dev


4. Install Express.js as a Dependency
Install Express.js and its types:

npm install express
npm install @types/express --save-dev

5. Set Up a Basic Express.js Server in TypeScript
Create a src directory and inside it, create an index.ts file:

mkdir src
touch src/index.ts
6. Update package.json Scripts

7. Run Your Project

npm start
